# Secure Login System

A secure web-based login system built using Flask and SQLite with password hashing and session management.

## Features
- User Registration
- Secure Login Authentication
- Password Hashing
- SQL Injection Protection
- Session Management
- Logout Functionality

## Technologies Used
- Python
- Flask
- SQLite
- Werkzeug Security

## Security Concepts Implemented
- Password Hashing
- Secure Authentication
- SQL Injection Prevention
- Session Handling
- Input Validation

## Educational Purpose
This project was developed to understand:
- Web application security
- Authentication systems
- Secure password storage
- Backend development
- Cybersecurity best practices
